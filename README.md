# Simulations of the ASG

This repository contains computational simulations of the **Ancestral Selection Graph (ASG)** developed as part of an undergraduate thesis in mathematics.

## 📌 Description

The goal of this project is to simulate the dynamics of the process \( M_t^{(N)} \), incorporating:

- Coalescence events
- Branching (selection) events

These simulations provide intuition about the model and support the theoretical results derived in the thesis.

## ⚙️ Requirements

- Python 3.x
- numpy
- matplotlib

Optional:
- Google Colab (recommended)
